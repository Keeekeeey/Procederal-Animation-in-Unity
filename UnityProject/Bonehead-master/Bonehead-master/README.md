<h4 align="center">
  A procedurally animated bearded dragon. Followed the [this tutorial](
https://weaverdev.io/blog/bonehead-procedural-animation). for the base code, edited and added to it as well as created the beardied dragon and mealworm models. 
  Sand textures were purchased from the Unity Asset Store [here](https://assetstore.unity.com/packages/2d/textures-materials/hand-painted-textures-vol-16-sand-237447)
  Dessert props were downloaded from the Unity Asset Store [here](https://assetstore.unity.com/packages/3d/vegetation/plants/cactus-pack-11547)
  Particales were downloaded from the Unity Asset Store [here](https://assetstore.unity.com/packages/vfx/particles/polygonal-s-low-poly-particle-pack-118355)
